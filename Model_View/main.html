This animation shows how the rectum changes from day to day. To run the animation select a viewpoint and playback speed.

Radiotherapy treatment of prostate cancer typically takes place over a number of weeks.
Movement of the rectum from day to day might cause poor control of dose to the recturm and hence unwanted side effects.

The data has been collected from x-ray images and the location of the rectum on the images identified by hand.

The 3D model was created in Blender using the clinical data and exported as a MD2 dataset.
This html5/javascript web page uses the JSC3D API to render and parse the MD2 file.

Clinical data analysis by Dr Jessica Scaife. Graphics implementation by Sam Sutcliffe.
